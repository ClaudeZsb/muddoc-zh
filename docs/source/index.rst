欢迎来到Muddoc!
===================================

**Muddoc** 是由开发者和 focg（fully on-chain game） 构建者编写的 `Mud <https://mud.dev/>`_ 文档，
特别针对原生 `Solidity <https://docs.soliditylang.org/en/latest/>`_ 开发者。
我们认为 `Mud 官方文档 <https://mud.dev/introduction>`_ 对于原生 Solidity 开发者来说不是特别友好，
所以我们决定编写这份文档。

查看 :doc:`/basics/001_introduction_to_mud_framework` 部分，了解 Mud 的介绍和示例，
包括如何 :ref:`installation` mud。

.. note::

   这份文档暂时只关注 Mud 项目中智能合约的编写部分。
